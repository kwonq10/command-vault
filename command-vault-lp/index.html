PROBLEM

こんな経験、ありませんか？

コマンドを忘れる

使いたいスラッシュコマンドの名前を忘れて、毎回ドキュメントを開いて確認している

メモ管理が面倒

よく使うコマンドをメモ帳やNotionに貼り付けて自己流管理。コピペが2ステップかかる

チームで共有できない

チームで便利なコマンドを発見しても、Slackに貼るだけで埋もれてしまう

FEATURES

Command Deck が解決します

🔍

自動で説明を取得

コマンド名を入力するだけで、公式ドキュメントから説明文を自動取得。取得できない場合はモーダルで手動入力できます。

⚡

ワンクリックコピー

コマンド名をクリックするだけでクリップボードにコピー。Alt+C で呼び出してすぐ貼り付けられます。

🔄

複数PCで同期

同期コードを設定するだけで Firebase 経由でリアルタイム同期。自宅・会社・チームメンバーと同じリストを共有できます。

🗂️

アーカイブ・並び替え

使わなくなったコマンドはアーカイブして非表示に。ドラッグ&ドロップでよく使うコマンドを上位に並び替えできます。

HOW TO START

3ステップで使い始める

01

インストール

Setup.exe をダウンロードして実行。インストール完了後にアプリが自動起動します。

→
02

コマンドを入力

Alt+C でアプリを呼び出し、入力欄にスラッシュコマンド名を入力して Add を押します。

→
03

即使える

説明文が自動取得されてリストに登録完了。クリックでコピーして Claude Code に貼り付けるだけ。
