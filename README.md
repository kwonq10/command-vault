# Command収集器

Claude Code のスラッシュコマンドを収集・管理する Windows デスクトップアプリ。

## 使い方

1. コマンド名を入力して Add を押す
2. Web検索で説明が自動取得されて登録される
3. Alt+C で呼び出し / 非表示

## インストール

[Releases](../../releases) から
`Command-Vault-Setup-1.0.0.exe` をダウンロードして実行。

## 開発環境での起動

```bash
npm install
npm run electron
```

## ビルド

```bash
npm run dist
```
